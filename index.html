AR Exploration Game Network Programming and Unity Gameplay Programming — Work In Progress

This AR Game was an idea which I upheld due to my current interest into AR games, this project I am working with two other friends Ivan Tomovic and Sebastian Adamov which are both creating assets. On this project I am using Node.js and Typescript which is the server side component of the game, which uses Socket.IO and Mongo which is where all my data is stored and is currently running on the Google Cloud. For the game itself, we are using Unity as the there was a free Map API called Mapbox that any other engine didn’t have.

The vision for this project was inspired by World of Warcraft and Diablo but put into a real-world perspective and to give real world opportunities for community building and friendship making. This project is currently still in development.
